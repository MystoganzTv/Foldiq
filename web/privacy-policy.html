Privacy Policy

Effective date: May 23, 2026 · Last updated: May 23, 2026

🔒 The short version: Foldiq does not collect, transmit, or store any personal data — ever. All processing happens entirely on your Mac. The app makes zero network connections.

1. Who we are

Foldiq ("the app", "we", "our") is a macOS application developed by Enrique Padrón. If you have any questions about this privacy policy, you can contact us at enrique.padron853@gmail.com.

2. Information we collect

We collect no information whatsoever. Foldiq does not collect, record, transmit, or share any personal data, usage data, analytics, telemetry, crash reports, or any other form of information from your device or your photo library.

Specifically, Foldiq does not collect:

  • Your name, email address, or any account information
  • Your photos, videos, or any media files
  • File names, folder paths, or metadata from your library
  • Device identifiers, IP addresses, or location data
  • Usage statistics, feature interaction data, or crash logs
  • Any information about your Mac or operating system

3. How the app processes your files

Foldiq reads and reorganizes files stored on your Mac's local storage or any directly connected drive. This processing happens entirely on your device using Apple's built-in system frameworks (FileManager, ImageIO, AVFoundation).

At no point does Foldiq transmit any file content, metadata, or path information to any server, cloud service, or third party. The app has no backend, no API, and no network connections of any kind.

4. File access

Foldiq only accesses folders that you explicitly select using the macOS file picker or by dragging them onto the app. The app will not access, scan, or read any folder you have not deliberately chosen.

On macOS, file access is governed by the operating system's sandbox and security-scoped bookmarks mechanism. Foldiq stores bookmarks for selected folders to allow re-access within the same session. These bookmarks are stored locally on your device using macOS UserDefaults and are never transmitted anywhere.

5. Network access

Foldiq makes no network connections of any kind. There are no outbound requests, no analytics endpoints, no update checks, no crash reporting services, and no cloud integrations. The app functions entirely offline.

You can verify this using macOS's built-in Network monitor or Little Snitch — Foldiq will show zero outbound connections.

6. Third-party services

Foldiq does not integrate with, depend on, or communicate with any third-party services, SDKs, libraries, or APIs that collect data. There are no advertising networks, analytics providers, or tracking frameworks included in the app.

7. Data storage

The only data Foldiq stores locally on your Mac is:

  • App preferences — organization mode, folder name preference, and other settings you configure. Stored in macOS UserDefaults.
  • Undo manifests — a log of every file movement performed, stored as a local JSON or CSV file in a location you choose. This file stays entirely on your Mac and is never transmitted anywhere.
  • Security-scoped bookmarks — macOS file access tokens for folders you select. Stored in UserDefaults, used only to re-access the same folder within the app.

You can delete all of this data at any time by deleting the app and its associated preferences using macOS's standard uninstall mechanisms.

8. Children's privacy

Foldiq does not collect any information from anyone, including children under 13. Because we collect no data at all, there are no special considerations for children's privacy beyond our general policy of collecting nothing.

9. Changes to this policy

If we ever make material changes to this privacy policy, we will update the effective date at the top of this page. Because the app collects no data, any future changes would only reflect updates to how the app works locally on your device.
