How it works

Four steps from messy to organized

Foldiq never moves a single file until you've reviewed and confirmed every planned change.

1

Select your folder

Drag any folder — or multiple folders — onto Foldiq. It handles nested subfolders, ZIP archives, and every common media format.

2

Foldiq scans & analyzes

It reads the real EXIF capture date, GPS location, and filename for every file. Duplicates are detected using SHA-256 hashing.

3

Preview everything

A full table shows every planned file move — source, destination, date, size. Filter and inspect before anything changes.

4

Apply & undo anytime

Confirm and Foldiq reorganizes your library. Every run creates an undo manifest — one click restores everything.

Features

Built for real photo libraries

Whether you have 500 photos or 50,000, Foldiq handles it safely.

📅

EXIF-accurate dates

Foldiq reads the real capture date from metadata — not the file creation date. Photos shared via WhatsApp or Telegram stay in true chronological order.

🔍

Duplicate detection

Exact duplicates are found via SHA-256 hashing. Probable duplicates — same name, similar size, close timestamp — are flagged separately so you decide what to keep.

↩️

Full undo, always

Every organization run writes a complete undo manifest. Change your mind days later — one click moves every file back to its exact original location.

📂

Multiple folders at once

Select several source folders in one scan. Foldiq merges and organizes them together, removing duplicates across all sources.

🗜️

ZIP archive support

Drop in a ZIP backup and Foldiq extracts it on the fly, organizing the contents alongside your other media as if they were never compressed.

🌍

GPS location folders

In Smart Hybrid mode, Foldiq adds the city and state to folder names when GPS data is available — so you can browse by place, not just date.

Organization modes

Pick how you want your library structured

Choose the folder depth that fits your style — from simple yearly buckets to a full date + location hierarchy.
