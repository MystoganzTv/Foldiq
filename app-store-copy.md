## Description
*(4,000 characters max)*

Your photo library deserves better than a folder called "Misc 2019 backup copy (2)."

Foldiq scans any folder of photos and videos on your Mac — including ZIP archives — and automatically builds a clean, dated folder structure based on the actual date each photo was taken. No manual sorting. No cloud required. No subscriptions.

**How it works**

1. Select any folder, multiple folders, or a .zip archive containing your media.
2. Foldiq reads EXIF data, GPS metadata, and filenames to identify every file's true capture date.
3. You review every planned file movement in a full preview table — nothing moves until you say so.
4. Foldiq reorganizes your library and writes a complete undo log, so you can reverse every change.

**Smart organization**

Choose how your library is structured: by Year, Year & Month, Exact Date, GPS Location, or the recommended Smart Hybrid mode that combines all of the above. Foldiq uses the real EXIF capture date — not the file creation date — so your library stays in true chronological order even after files were copied, shared, or forwarded via WhatsApp or Telegram.

**Duplicate detection**

Foldiq finds exact duplicates using SHA-256 file hashing and routes them to a separate Duplicates folder for easy review. It also detects probable duplicates — files with the same name, nearly identical size, and similar timestamps — and flags them separately so you decide what to keep.

**Advanced filters & preview**

Before applying any changes, review every planned move in a searchable, sortable table. Filter by file type, date range, camera model, GPS availability, or duplicate status. See exactly where each file will go, inspect thumbnails, or drill into the full folder tree.

**Full undo, always**

Every organization run creates a detailed undo manifest. If you change your mind, one click moves every file back to its exact original location — even days later.

**What Foldiq organizes**
Photos: JPG, JPEG, PNG, HEIC, TIFF, and RAW formats including CR2, NEF, ARW, and DNG.
Videos: MOV, MP4, and M4V.
Archives: .zip files are automatically extracted and their contents organized alongside your other media.

**Privacy first**

All processing happens locally on your Mac. Foldiq never connects to the internet, never uploads your photos anywhere, and never asks for a cloud account. Your library stays yours.
